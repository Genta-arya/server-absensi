API Documentation

Update Avatar

Endpoint:

PUT /api/v1/profile/avatar/:id

Deskripsi:

Endpoint ini digunakan untuk memperbarui foto profil pengguna. Mengharuskan pengguna untuk mengupload file gambar sebagai avatar baru.

Parameter:

  • id (path parameter): ID pengguna yang foto profilnya akan diperbarui.

Request Body:

Menggunakan Form Data untuk mengupload file gambar.

{
    "avatar": "file (gambar)"
}

Response:

{
    "message": "Avatar updated successfully.",
    "avatar": "URL avatar baru"
}

Jika terjadi kesalahan, response akan memberikan pesan kesalahan yang sesuai.

Contoh Penggunaan:

curl -X POST \
    -F "avatar=@/path/to/your/image.jpg" \
    https://dev-absensi.hkks.shop/api/v1/profile/avatar/123

Silakan gunakan endpoint di atas untuk memperbarui avatar pengguna Anda.
