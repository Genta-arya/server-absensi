API Documentation

Update Profile

Endpoint:

PUT /api/v1/profile/:id

Deskripsi:

Endpoint ini digunakan untuk memperbarui informasi profil pengguna, termasuk nama dan email.

Parameter:

  • id (path parameter): ID pengguna yang informasinya akan diperbarui.

Request Body:

{
    "name": "Nama Lengkap",
    "email": "example@example.com"
}

Format JSON untuk memperbarui informasi nama dan email.

Response:

{
    "message": "Profile updated",
    "user": {
        "id": "123",
        "name": "Nama Lengkap",
        "email": "example@example.com",
        "avatar": "URL avatar"
    }
}

Jika terjadi kesalahan, response akan memberikan pesan kesalahan yang sesuai.

Contoh Penggunaan: 

curl -X PUT \
    -H "Content-Type: application/json" \
    -d '{"name": "Nama Lengkap", "email": "example@example.com"}' \
    https://dev-absensi.hkks.shop/api/v1/profile/123

Silakan gunakan endpoint di atas untuk memperbarui profil pengguna Anda.
