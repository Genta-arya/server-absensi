API Documentation - Logout

Logout Pengguna

Endpoint untuk mengeluarkan pengguna dari sesi aktif.

POST /logout

Mengeluarkan pengguna dengan ID yang diberikan.

Request Body:


{
    "id": "string"  // ID pengguna yang akan dikeluarkan
}

Response:

200 OK: Jika logout berhasil.


{
    "message": "Berhasil logout"
}

400 Bad Request: Jika ID tidak disertakan dalam permintaan.


{
    "message": "Mohon lengkapi id" // Jika ID tidak ada
}

404 Not Found: Jika pengguna dengan ID tersebut tidak ditemukan.


{
    "message": "User not found" // Jika ID tidak valid
}

Penggunaan

Untuk mengeluarkan pengguna, kirimkan permintaan POST ke /logout dengan body yang berisi ID pengguna yang akan dikeluarkan.
