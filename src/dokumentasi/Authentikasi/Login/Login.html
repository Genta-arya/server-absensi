API Documentation - Login

Login Pengguna

Endpoint untuk login pengguna dengan menggunakan NIM dan password.

POST /login

Login pengguna dengan memberikan data berikut:

Request Body:


{
    "nim": "string",  // NIM pengguna
    "password": "string"  // Password pengguna
}

Response:

200 OK: Jika login berhasil.


{
    "message": "Login berhasil",
    "data": {
        "user": {
            "id": "string",  // ID pengguna
            "nim": "string",  // NIM pengguna
            "name": "string | null",  // Nama pengguna (jika ada)
            "email": "string | null",  // Email pengguna (jika ada)
            "isLogin": false,  // Status login
            "token": null,  // Token (jika ada)
            "avatar": "string | null",  // URL Avatar (jika ada)
            "role": "MHS" | "DOSEN" | "ADMIN"  // Role pengguna
        },
        "token": "string"  // Token JWT untuk autentikasi selanjutnya
    }
}

400 Bad Request: Jika kolom NIM atau password tidak lengkap.


{
    "message": "Mohon lengkapi nim" // Jika NIM kosong
}
// atau
{
    "message": "Mohon lengkapi password" // Jika password kosong
}

401 Unauthorized: Jika NIM atau password tidak sesuai.


{
    "message": "NIM atau password salah"
}

Penggunaan

Untuk login, kirimkan permintaan POST ke /login dengan body sesuai dengan yang dijelaskan di atas. Jika login berhasil, respons akan berisi token JWT yang bisa digunakan untuk autentikasi pada permintaan berikutnya.
