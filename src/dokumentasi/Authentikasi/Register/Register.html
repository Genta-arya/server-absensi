API Documentation - Register

Register Pengguna

Endpoint untuk mendaftar pengguna baru.

POST /register

Mendaftar pengguna baru dengan memberikan data berikut:

Request Body:


{
    "nim": "string",  // NIM pengguna (harus unik)
    "password": "string",  // Password pengguna
    "role": "MHS" | "DOSEN"  // Role pengguna
}

Response:

201 Created: Jika pendaftaran berhasil.


{
    "message": "Pendaftaran berhasil",
    "data": {
        "id": "string",  // ID pengguna
        "nim": "string",  // NIM pengguna
        "role": "MHS" | "DOSEN" // Role pengguna
    }
}

400 Bad Request: Jika kolom tidak lengkap atau role tidak valid. 


{
    "message": "Mohon lengkapi semua kolom" // Jika ada kolom yang kosong
}
// atau
{
    "message": "Role tidak valid. Pilih antara MHS atau DOSEN" // Jika role tidak valid
}

409 Conflict: Jika NIM sudah terdaftar.


{
    "message": "NIM sudah terdaftar"
}

Penggunaan

Untuk mendaftar pengguna baru, kirimkan permintaan POST ke /register dengan body sesuai dengan yang dijelaskan di atas.
